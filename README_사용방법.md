# 📌 광고주별 완료건 삭제 SQL 사용 방법

## 사용 순서

각 단계별로 파일을 **하나씩** 실행하세요!

### 1️⃣ 0단계: 전체 광고주 목록 보기
- 파일: `0단계_전체광고주목록보기.sql`
- 설명: 광고주명을 모를 때 먼저 실행
- 수정 필요: 없음 (그대로 실행)
- 결과: 전체 광고주 목록이 표시됩니다

### 2️⃣ 1단계: 광고주 검색하기
- 파일: `1단계_광고주검색하기.sql`
- 설명: 특정 광고주를 검색합니다
- 수정 필요: `'%검색어_여기입력%'` 부분을 실제 광고주명으로 변경
- 예시: `'%테스트%'` 또는 `'%홍길동%'`
- 결과: 검색된 광고주의 ID와 정보가 표시됩니다

### 3️⃣ 2단계: 완료건 목록 보기
- 파일: `2단계_완료건목록보기.sql`
- 설명: 해당 광고주의 완료건 목록을 확인합니다
- 수정 필요: `'%검색어_여기입력%'` 부분을 1단계와 동일하게 입력
- 결과: 삭제할 주문의 **주문ID**를 확인하세요 (긴 문자열)

### 4️⃣ 3단계: 완료건 삭제하기
- 파일: `3단계_완료건삭제하기.sql`
- 설명: 선택한 완료건을 삭제합니다
- 수정 필요: `'주문ID_여기입력'` 부분을 2단계에서 복사한 주문ID로 변경
- ⚠️ 주의: 삭제는 되돌릴 수 없습니다!

## 예시

1. 0단계 실행 → 전체 광고주 목록 확인
2. 1단계에서 `'%홍길동%'` 입력 → 광고주 ID 확인 (예: `12345678-1234-1234-1234-123456789abc`)
3. 2단계에서 `'%홍길동%'` 입력 → 주문 목록 확인, 주문ID 복사 (예: `98765432-5678-5678-5678-987654321def`)
4. 3단계에서 주문ID 입력 → 삭제 실행

## 문제 해결

### 광고주 검색 결과가 안 나올 때
- 0단계로 전체 목록 확인
- 1단계 파일의 하단 디버깅 쿼리 사용 (완료건이 있는 광고주만 보기)

### 완료건 목록이 안 나올 때
- 2단계 파일의 "방법 2" 사용 (광고주 ID로 직접 검색)

### 에러 발생 시
- 주문ID가 작은따옴표 안에 있는지 확인 (`'주문ID'`)
- 주문ID에 앞뒤 공백이 없는지 확인
- 주문ID를 정확히 복사했는지 확인
