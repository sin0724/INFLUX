## 2️⃣ 관리자 계정 생성

배포된 환경에서 관리자 계정을 생성해야 합니다.

### 방법 1: Supabase SQL Editor 사용 (권장)

1. **Supabase 대시보드** 접속
   - https://supabase.com/dashboard
   - 프로젝트 선택

2. **SQL Editor** 열기
   - 좌측 메뉴에서 "SQL Editor" 클릭
   - "New query" 클릭

3. **관리자 계정 생성 SQL 실행**

먼저 비밀번호 해시를 생성해야 합니다. 로컬에서:

```bash
node -e "const bcrypt = require('bcryptjs'); console.log(bcrypt.hashSync('1234', 10));"
```

또는 아래 스크립트를 사용:

```sql
-- 관리자 계정 생성
-- 비밀번호: 1234 (bcrypt 해시 필요)
-- 먼저 로컬에서 해시 생성 후 사용하세요

-- 임시로 직접 해시 생성 (로컬 스크립트 권장)
INSERT INTO users (username, password, role, "totalQuota", "remainingQuota", "isActive")
VALUES (
  'admin1',
  '$2a$10$rOzJ0X5HnHc5KqV9z0Kz9eH5KqV9z0Kz9eH5KqV9z0Kz9eH5KqV9z0K',  -- 임시 해시 (로컬에서 생성 권장)
  'admin',
  0,
  0,
  true
)
ON CONFLICT (username) DO NOTHING;
```

**더 안전한 방법**: 로컬에서 `scripts/create-test-accounts.js` 실행

### 방법 2: 로컬 스크립트 실행

로컬 컴퓨터에서:

```bash
# .env.local 파일에 환경 변수 설정 필요
# Supabase 연결 정보 입력

node scripts/create-test-accounts.js
```
